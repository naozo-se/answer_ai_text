それぞれのAIモデルの進化に伴い、現在では単なるテキストだけでなく、多種多様なフォーマットをそのままアップロードして処理できるようになっています。

### 各サービスの特徴と補足

**1. Anthropic (Claude)**
テキストやドキュメント、ソースコードなどの「文字情報」の処理に特化しています。長文の文脈を破綻なく理解する能力が非常に高く、複数のPDFや長大なコードを投げ込んでの高精度な分析が得意です。
* **(※1) PPTX対応の強化:** 2026年初頭のアップデート（Opus 4.6など）により、`.pptx`ファイルの読み込みや、PowerPointとしてそのまま編集できるネイティブなスライド生成能力が大幅に強化されました。
* **弱点:** 標準のWebチャットUIでは、現時点でも動画や音声ファイルの直接アップロードには対応していません。

**2. OpenAI (ChatGPT)**
「Advanced Data Analysis（旧Code Interpreter）」という強力なPythonコード実行環境を裏側で持っているため、事実上「プログラムで処理・変換できるファイルなら何でも対応できる」のが最大の強みです。
* **(※3) 動画への対応:** アップロードした動画から特定のフレームを切り出したり、視覚モデル（GPT-4o等）で内容を分析したりすることが可能です。
* **(※5) 圧縮ファイルの解凍:** `.zip`などをアップロードしてAI自身に解凍させ、中に入っている複数のファイルを一括で処理・分析させることができます。

**3. Google (Gemini) ※私です**
Google Workspace（ドキュメントやスプレッドシート等）とのシームレスな連携はもちろん、**「動画と音声のネイティブ理解」**において群を抜いています。
* **(※2) 幅広い画像対応:** Apple製デバイスで標準的に使われる `.heic` や `.heif` 画像の直接アップロードにも対応しています。
* **(※4) 強力なメディア解析:** 動画（有料プランでは最大1時間）や音声ファイルを直接アップロードし、テキストと同じ感覚で「この動画の〇〇について要約して」といった指示を高精度に処理できます。
* **(※6) コード群の一括読み込み:** 現在は`.zip`ファイルの直接アップロードにも対応しており、複数のファイルが格納されたコード群などを一気に読み込ませることが可能です（※ファイル数等の上限あり）。

> **💡 補足事項:** > 上記は標準的なWebブラウザ版・アプリ版（有料プランの最新モデルを含む）での対応状況です。無料プランでお使いの場合、一部のファイル形式や、1回あたりのファイルサイズ上限・アップロード数に制限がかかる場合があります。

動画の解析手順や、大容量のエクセルデータの処理など、特定のファイル形式の扱い方についてさらに詳しく知りたいものはございますか？
